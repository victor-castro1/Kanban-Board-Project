# Kanban Board - Quadro de Tarefas

## O que é isso aqui?

É um **Quadro Kanban** bem simples e direto. Basicamente, é como aquelas anotações com Post-its que você cola na parede, mas digitalmente. Você organiza suas tarefas em 3 colunas:

- **A Fazer** - Tarefas que ainda precisa começar;
- **Fazendo** - As tarefas que você está fazendo **AGORA**;  
- **Concluído** - Aquelas tarefas que você finalmente terminou!;

## Como usar? 

1. **Clone o repositório** para sua máquina
2. Abra o arquivo `index.html` no seu navegador
3. Pronto! Seu quadro está rodando

## O que você consegue fazer?

1. **Adicionar tarefas** - Clique no botão "+ Novo Card" e digite a tarefa na qual precisa fazer;
2. **Arrastar tarefas** - Puxa o card com o mouse e solta na coluna seguinte;
3. **Editar tarefas** - Dá um duplo clique no card para editar o texto;
4. **Deletar tarefas** - Clica no "X" do card caso queira deletar a tarefa;
4. **Tudo fica guardado** - O progresso é salvo automaticamente no computador.

## Regras de Negócio:

- **Máximo 3 tarefas na coluna "Fazendo"** - Para você não sobrecarregar o quadro e "poluir" visualmente
- **Total de 20 tarefas no quadro** - Limite colocado para não virar um desastre  
- **Tarefas precisam de pelo menos 3 letras** - Nada de "oi" ou "ok".  
- **Não pode voltar pra trás** - Uma vez que avança de coluna, segue em frente. 
- **Sem tarefas iguais na mesma coluna** - Não vale duplicar

## Tecnologias usadas

Bem simples mesmo:
- **HTML** - A estrutura das coisas;
- **CSS** - Para ficar visualmente bonito e responsivo;
- **JavaScript puro** - Não há uma biblioteca externa;
