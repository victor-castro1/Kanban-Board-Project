# Kanban Board - Seu Quadro de Tarefas 📋

Oi! 👋 Se você está aqui, provavelmente está cansado de ficar anotando tarefas em papel ou em um monte de abas abertas no navegador, né? Bem, esse projeto é para você!

## O que é isso aqui?

É um **Quadro Kanban** bem simples e direto. Basicamente, é como aquelas anotações com Post-its que você cola na parede, mas digital e muito melhor! Você organiza suas tarefas em 3 colunas:

- **A Fazer** - As tarefas que você ainda precisa começar
- **Fazendo** - As tarefas que você está trabalhando AGORA  
- **Concluído** - Aquelas tarefas que você finalmente terminou! 🎉

## Como usar? É fácil demais!

1. **Clone o repositório** para sua máquina
2. Abra o arquivo `index.html` no seu navegador
3. Pronto! Seu quadro está rodando

## O que você consegue fazer?

✅ **Adicionar tarefas** - Clique no botão "+ Novo Card" e digite o que precisa fazer  
✅ **Arrastar tarefas** - Puxa o card com o mouse e solta na coluna que quiser  
✅ **Editar tarefas** - Dá um duplo clique no card para mudar o texto  
✅ **Deletar tarefas** - Clica no "X" do card se mudar de ideia  
✅ **Tudo fica guardado** - Seu progresso é salvo automaticamente no navegador  

## Regras do jogo (porque nem tudo é permitido 😄)

- **Máximo 3 tarefas na coluna "Fazendo"** - Para você não se sobrecarregar  
- **Total de 20 tarefas no quadro** - Pra não virar um caos  
- **Tarefas precisam de pelo menos 3 letras** - Nada de "oi" ou "ok"  
- **Não pode voltar pra trás** - Uma vez que avança de coluna, segue em frente!  
- **Sem tarefas iguais na mesma coluna** - Não vale duplicar

## Tecnologias usadas

Bem simples mesmo:
- **HTML** - A estrutura das coisas
- **CSS** - Para ficar bonito e responsivo
- **JavaScript puro** - Toda a mágica acontece aqui, sem bibliotecas externas
